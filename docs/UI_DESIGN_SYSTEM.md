# YuxTrans UI/UX 设计系统 —— 书房衬纸版

> 版本：v2.0（基于浏览器扩展 0.4.0 全面重写）  
> 核心气质：阅读优先，翻译隐身。不是效率工具，而是铺在网页边缘的虚拟衬纸。

---

## 1. 设计哲学

### 1.1 一句话原则
**让用户忘记自己正在使用一个“工具”。**

翻译不是对原文的覆盖，而是页边悄然生长的注脚；UI 不是弹出的窗口，而是书页旁自然延伸的空白。所有视觉决策服从文本可读性与阅读节奏的连续性。

### 1.2 气质锚点
温和 · 克制 · 陪伴 · 深阅读

### 1.3 情绪关键词
安静陪伴 · 深度阅读 · 克制温暖 · 语言连接 · 长久信赖 · 书房黄昏 · 页边批注 · 纸墨温度

## 2. 色彩体系

色彩从 Logo 的墨韵色延伸，全局饱和度控制在 15% 以下，像被时间洗褪过的印刷品。

### 2.1 主色盘（浅色模式 / 纸张模式）

| 角色 | 色值 | 意象 | 使用场景 |
|------|------|------|----------|
| **墨韵** | `#2C2825` | 温润炭黑 | 主文本、图标、关键边框、激活态 |
| **书页** | `#F5F1EA` | 暖纸白 | 面板底色、浮层背景、输入框底色 |
| **批注** | `#9E968A` | 温和灰褐 | 次要文本、禁用态、分割线、占位符 |
| **暮瞳** | `#B8A5C4` | 淡薰衣草灰 | 悬停光晕、当前句高亮边缘、选中态微光 |
| **留白** | `transparent` | 无 | 面板与网页之间的呼吸间隙 |

### 2.2 色彩原则

- 任何面板、浮层必须使用半透明毛玻璃：`rgba(245, 241, 234, 0.92)` + `backdrop-filter: blur(12px)`，让底层网页隐约透过来。
- 阴影不使用纯黑，使用墨韵色极低透明度：`rgba(44, 40, 37, 0.06)` 至 `0.12`，模拟纸张叠压厚度。
- `#000000` 与 `#FFFFFF` 绝不出现。
- 状态色（成功/警告/错误）也必须使用低饱和变体，避免破坏纸张氛围。

### 2.3 深色模式：黄昏书房

不是“反色”，而是“黄昏书房”。

| 角色 | 色值 |
|------|------|
| 底色 | `#1A1816` |
| 面板底色 | `rgba(26, 24, 22, 0.92)` |
| 主文本 | `#D4CFC7` |
| 次要文本 | `#9E968A` |
| 书页叠压 | `#2A2622` |
| 暮瞳微光 | `#B8A5C4`（在暗处更明显） |

## 3. 字体系统

### 3.1 字体栈

```css
/* 正文与阅读文本 */
--yxt-font-serif: 'Source Serif 4', 'Literata', 'Merriweather', 'Source Han Serif SC', 'Noto Serif SC', 'FangSong', 'STSong', serif;

/* 标签、元信息、按钮 */
--yxt-font-sans: 'Inter', 'Suisse Intl', -apple-system, BlinkMacSystemFont, 'PingFang SC', 'Microsoft YaHei', sans-serif;

/* 中文楷体/书宋场景（设置页大标题、引用） */
--yxt-font-cjk: 'Source Han Serif SC', 'FZShuSong-Z01S', 'LXGW WenKai', 'STKaiti', 'KaiTi', serif;
```

### 3.2 排版尺度

| 层级 | 字号 | 行高 | 字重 | 字间距 | 说明 |
|------|------|------|------|--------|------|
| 正文/译文 | 14-15px | 1.75 | 400 | normal | 与网页原文严格保持一致 |
| 原文引用 | 12-13px | 1.6 | 400 | normal | 斜体，批注色 |
| 标签/元信息 | 11-12px | 1.4 | 400 | 0.02em | 禁用全大写 |
| 面板标题 | 16-18px | 1.3 | 500 | -0.01em | 像章节标题 |
| 小标题 | 13-14px | 1.4 | 500 | normal | 表单分区 |

### 3.3 翻译结果标识

- 不放大、不加粗、不变色。
- 左侧以 `2px` 宽的暮瞳色竖线作为“批注栏”标识。
- 例句/引用段落同样使用左侧暮瞳色竖线。

## 4. 布局与空间

### 4.1 网格系统

不采用刚性像素网格，采用「排版基线网格」。垂直间距遵循 `8px` 的温和倍数，允许 `4px` 微调。

### 4.2 层级逻辑

| 层级 | 名称 | 遮挡上限 | 示例 |
|------|------|----------|------|
| 0 | 网页本身 | 0% | 永远主角 |
| 1 | 衬纸层 | 不遮挡主体 | 翻译结果、单词释义 |
| 2 | 工具层 | 可拖拽移开 | 划词操作条、设置面板 |
| 3 | 系统层 | 独立小册子 | 扩展弹窗、全局设置 |

### 4.3 圆角规范

- 面板与大卡片：`6px-8px`（便签纸感）
- 按钮与输入框：`4px`（铅字倒角）
- 标签与徽标：`2px`（几乎不可察觉）
- 禁止 `> 12px` 圆角与 `9999px` 胶囊

### 4.4 边框

- 默认：`1px solid rgba(44, 40, 37, 0.08)`（纸张边缘厚度）
- 激活：`1px solid rgba(184, 165, 196, 0.4)`（暮瞳微光渗透）

## 5. 核心场景设计标注

### 5.1 场景一：划词悬浮操作条

**定位：** 第 2 层工具层，可拖拽移开（v2.1 可选增强，v2.0 保证可关闭）。

**形态：**
- 不是胶囊按钮，而是一张轻薄的「标签纸」。
- 尺寸：高度 `32px-36px`，内边距 `6px 12px`。
- 底色：`rgba(245, 241, 234, 0.95)` 毛玻璃。
- 边框：`1px solid rgba(44, 40, 37, 0.08)`。
- 阴影：`0 2px 8px rgba(44, 40, 37, 0.06)`。
- 圆角：`6px`。

**内容：**
- 仅保留 2-3 个最高频操作：翻译 / 朗读 / 收藏。
- 图标 `16px`，线性，线宽 `1.5px`。
- 图标旁配极小文字标签 `11px`，批注色。

**出现动效：**
- 从选中文本下方 `8px` 处浮现。
- `opacity: 0 → 1`，`translateY(4px) → 0`。
- 时长 `200ms`，`ease-out`，像被风吹起的纸片轻轻落下。

**消失动效：**
- 鼠标离开选区 `800ms` 后，`opacity` 缓降至 `0`。
- 时长 `200ms`，`ease-in`。

### 5.2 场景二：翻译结果面板

**定位：** 第 1 层衬纸层，贴合在文本边缘的书页批注。

**形态：**
- 宽度：`280px-320px`。
- 底色：`rgba(245, 241, 234, 0.92)` 毛玻璃 + `backdrop-filter: blur(12px)`。
- 边框：左边缘 `2px` 暮瞳色竖线 + 其余三边 `1px rgba(44, 40, 37, 0.08)`。
- 阴影：`0 12px 32px rgba(44, 40, 37, 0.08), 0 2px 8px rgba(44, 40, 37, 0.04)`。
- 圆角：`6px`（右上/右下/左下），左上角 `2px`（像便签撕口）。

**排版：**
- 原文：`13px`，斜体，批注色，行高 `1.6`，底部 `8px` 间距。
- 译文：`15px`，正体，墨韵色，行高 `1.75`。
- 词性标注：`10px`，批注色，像字典轻声注释。
- 例句：`13px`，行高 `1.6`，左侧带 `2px` 暮瞳色竖线。
- 引擎标签：`10px`，批注色，不加粗，无背景填充。

**关闭：**
- 右上角 `×` 图标按钮 `28px × 28px`。
- hover 时出现暮瞳色微光，不放大。
- 支持 `Esc` 关闭。

**加载态：**
- 不显示旋转 Spinner。
- 显示批注色省略号 `…`（`font-size: 18px`，`letter-spacing: 0.1em`），像未写完的批注。

### 5.3 场景三：设置页

**定位：** 第 3 层系统层，从扩展图标向下展开的「折页」。

**整体结构：**
- 宽度：`360px-900px`（options.html 全页最大 `900px`）。
- 最大高度：`80vh`，可滚动。
- 布局：左侧 `220px` 侧边目录 + 右侧主内容。
- 底色：侧边栏与主内容均使用书页色，但主内容略浅（`#FAF8F3`），像翻开的书页。

**侧边栏：**
- 标题 `YuxTrans`：`20px`，`--yxt-font-cjk`，墨韵色，字重 `500`。
- 副标题 `Configuration`：移除或改为「设置」`12px` 批注色。
- 导航项：`14px`，墨韵色，无全大写。
- 激活态：左侧 `2px` 暮瞳色竖线 + 背景 `rgba(184, 165, 196, 0.08)`。
- hover：背景 `rgba(44, 40, 37, 0.03)`。

**主内容分区：**
- 章节标题：`16px`，墨韵色，字重 `500`，底部 `12px` 间距。
- 章节之间使用 `1px solid rgba(44, 40, 37, 0.06)` 分隔线。
- 不使用卡片、不使用阴影区块、不使用大圆角。

**表单控件：**
- 输入框/选择框：无明确边框，底部 `1px` 批注色下划线，聚焦时过渡为暮瞳色。
- 背景：透明或书页色。
- 占位符：批注色，像铅笔草稿淡痕。
- 标签：`12px`，墨韵色，字重 `400`，`margin-bottom: 8px`。

**开关组件：**
- 不是 iOS 胶囊，而是活字印刷的「嵌入/凸出」质感。
- 外框：`28px × 16px`，`1px solid rgba(44, 40, 37, 0.2)`，圆角 `4px`。
- 滑点：`12px × 12px` 小圆点，墨韵色。
- 关闭态：滑点居左，背景书页色。
- 开启态：滑点居右，背景 `rgba(184, 165, 196, 0.2)`。

**按钮：**
- 主按钮：墨韵色文字 + 书页色底色 + `1px` 墨韵色边框。
- hover：底色 `#EDE8DF`（像手指轻触纸张温度）。
- 次按钮：纯文字，批注色，hover 文字变墨韵色。

**翻译风格选项：**
- 由胶囊/卡片改为「纸张选项」。
- 单选卡片：书页底色，细边框，圆角 `4px`。
- 选中态：左侧 `2px` 暮瞳色竖线 + 背景 `rgba(184, 165, 196, 0.08)`。

### 5.4 场景四：扩展弹窗（Popup）

**定位：** 第 3 层系统层，像从书桌上拿起的一本独立小册子。

**整体：**
- 宽度：`320px`。
- 底色：`rgba(245, 241, 234, 0.98)`（弹窗不需要太多底层透传，但仍保持暖纸色）。
- 边框：`1px solid rgba(44, 40, 37, 0.08)`。
- 圆角：`8px`。
- 内边距：`20px`。

**Header：**
- 品牌名 `YuxTrans`：`18px`，`--yxt-font-cjk`，墨韵色，字重 `500`。
- 版本号：`10px`，批注色，不加粗。
- 设置按钮：图标 `16px`，批注色，hover 墨韵色。

**状态行：**
- 状态点：`6px` 圆点，颜色随状态变化（检测中-批注色，正常-低饱和绿 `#7A9E8A`，错误-低饱和红 `#B87878`）。
- 文本：`11px`，批注色。

**模型选择：**
- 下划线输入框样式，箭头图标 `10px` 批注色。

**主操作按钮：**
- 墨韵色文字 + 书页色底色 + `1px` 墨韵色边框。
- 全宽，圆角 `4px`，`padding: 11px`。
- hover 底色 `#EDE8DF`。

**统计行：**
- 三列均分，顶部与底部分隔线 `1px rgba(44, 40, 37, 0.06)`。
- 数值：`16px`，墨韵色，`font-variant-numeric: tabular-nums`。
- 标签：`10px`，批注色。

**流式开关：**
- 使用新版活字印刷开关（同设置页）。

**Toast：**
- 底部中央浮现，毛玻璃书页色，批注色文字，无黑色背景。

## 7. 动效系统

### 7.1 时间哲学

所有动效像翻书页一样——有重量感，但不停滞。

### 7.2 曲线与时间

| 用途 | 曲线 | 时长 |
|------|------|------|
| 浮现 | `ease-out` | `200-300ms` |
| 消失 | `ease-in` | `150-200ms` |
| 位移/滑动 | `cubic-bezier(0.25, 0.1, 0.25, 1.0)` | `250ms` |
| 高亮扫过 | `ease` | `300ms` |
| 悬停反馈 | `ease` | `150-200ms` |

### 7.3 具体动效

- **划词操作条浮现**：`opacity 0→1 + translateY(4px)→0`，`200ms ease-out`
- **划词操作条消失**：`opacity 1→0`，`200ms ease-in`，延迟 `800ms`
- **翻译面板浮现**：`opacity 0→1 + translateY(8px)→0`，`250ms ease-out`
- **当前句高亮**：背景从 `transparent` 到 `rgba(184, 165, 196, 0.08)`，`300ms ease`
- **按钮悬停**：背景色过渡 `200ms ease`
- **图标按钮悬停**：`scale 1→1.05 + 背景微光`，`200ms ease`

### 7.4 禁止的动效

- 不使用旋转 Spinner
- 不使用 Skeleton 骨架屏
- 不使用弹性/弹跳动画
- 不使用 Material Design 涟漪
- 不使用霓虹/发光/过度毛玻璃

## 9. 浏览器扩展特定场景

### 9.1 扩展图标

- 使用 Logo 墨韵色单色版本，在深色/浅色工具栏均保持识别度。
- 激活态（当前页面翻译开启）：图标下方出现 `2px` 宽暮瞳色微光横线。

### 9.2 右键菜单

- 不粗暴覆盖浏览器默认样式，而是在菜单项中注入品牌字体与色彩。
- hover 时背景变为书页色，文字保持墨韵色。

### 9.3 网页内容注入

- 所有注入 DOM 元素必须使用 Shadow DOM 隔离样式（当前已实现），但视觉上必须「长在该网页里」。
- `z-index` 严格控制在合理范围，不遮挡原生弹窗、视频控件。
- 深色网页自动切换为黄昏模式。
