# ADR 0005: Options 五模块信息架构与分栏保存 / 设置页氛围与分区点缀

## 状态
Accepted

## 背景
Options「翻译偏好」单 tab 聚合了语言风格、划词交互、显示、术语表、缓存、站点规则与危险操作，扫视成本高、心智混杂。视觉上大面积单色书页底缺少层次，易引发审美疲劳。grill 会话锁定了可执行的 IA 与视觉边界，且 ActiveConfig 仍须保持**一份**存储对象（见 ADR 0002）。

## 决策

### 1. 五顶栏模块（A2）
| Tab id | 中文 | 职责 |
|--------|------|------|
| `profiles` | 服务档案 | ProviderProfile 配置与档案管理（角色不变） |
| `preference` | 翻译偏好 | 「译什么」：当前档案锚点、按需上手三步、离线模式、语言方向、翻译风格 |
| `interaction` | 交互与显示 | 「怎么触发 / 长什么样」：触发、流式、原文样式、悬停、词典 + 进阶项 |
| `data` | 数据与存储 | 术语表、缓存、导入导出、网站规则/故障转移、危险清除 |
| `diagnostics` | 诊断排障 | 只读观测看板与日志，不写 ActiveConfig |

### 2. 分栏保存（Module Save / F1）
- 可写模块 `preference` / `interaction` / `data` 各自底栏保存**本栏字段**；通过字段映射切片写入同一 `setConfig`，不是三套配置。
- `profiles` 维持既有档案保存流；`diagnostics` 无配置保存。
- 明确不做：全局「保存全部」、改完即自动保存、离开 tab 脏检查弹窗（F4 延后）。

### 3. 交互默认 vs 进阶（B1）
- **默认可见**：触发方式、流式输出、整页原文样式、悬停段落翻译、单词词典模式。
- **进阶折叠**：自动复制、悬停修饰键、双击查词、输入框翻译、正文区域识别、双档案对照。
- 父关子灰显（悬停→修饰键；词典→双击）。

### 4. 上手区可见性（G1）
- 保留「当前使用」卡片（跳转服务档案）与离线模式在翻译偏好。
- 「快速开始」三步仅在 `firstRunPending` 或无可用档案（无 profiles / 无有效 activeProfile）时显示。

### 5. 设置页氛围（Options Atmosphere / D1）
- 仅 Options：静态纸纹 + **一层** CSS 缓漂径向光晕（~30s，`transform` 动画）；禁止视频/Canvas/粒子/重滤镜。
- `prefers-reduced-motion: reduce` 时停止运动。

### 6. 分区点缀色（Zone Accent / E1）
- 仅导航与块标题等 chrome 使用极低饱和色相提示，缓解单色扫视疲劳。
- 不抬高全局品牌主色饱和度；不做风格卡独立配色（E4）；**本轮不改** Popup / 划词浮层视觉。

### 7. 纯函数归属
模块字段表、分栏 pick、上手区是否显示等逻辑放在 `extension/lib/product-helpers.js`，供 Options 与单测共用。

## 后果
- Options 顶栏由 3 增至 5；`data-tab="general"` 退役为 `preference` / `interaction` / `data`。
- 用户跨模块改多项需分别保存；漏保存风险用文案标明「本页」。
- 设计令牌可增 Options 氛围/分区变量；`UI_DESIGN_SYSTEM` 宜补短注以免被当成全局反色改造。

## 拒绝的替代方案
- **同 tab 仅加间距/折叠（C）**：无法拆开「译什么 / 怎么用 / 数据」的心智。
- **合并诊断进数据（A3）**：只读观测与可写/危险操作混页，易误触。
- **全局自动保存（F3）**：设置页含限额、域名列表与危险旁路，误触代价高。
- **摄影/视频背景或全局抬饱和（C3/E3）**：违背书房衬纸与阅读优先。
