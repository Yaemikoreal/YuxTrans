# Qwenfy - AI 翻译微型软件与浏览器插件

> 追求极致响应速度与翻译准度的轻量级 AI 翻译工具

---

## 🎯 项目目标

打造一款**响应速度极快、翻译质量精准**的 AI 翻译工具，覆盖桌面端和浏览器场景。

### 核心指标

| 指标 | 目标值 |
|------|--------|
| **响应速度** | < 500ms（本地模型） / < 2s（云端 API） |
| **翻译准度** | BLEU Score > 0.75，人工评分 > 4.2/5 |
| **内存占用** | < 200MB（桌面端） |
| **启动时间** | < 3s |

## 🚀 快速开始

### 环境要求

- Python 3.10+
- Node.js 18+ (浏览器插件)
- Ollama (本地模型)

### 安装

```bash
# 克隆仓库
git clone https://github.com/Yaemikoreal/qwenfy.git
cd qwenfy

# 安装依赖
pip install -r requirements.txt

# 启动桌面客户端
python -m qwenfy.desktop
```
